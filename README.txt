Claudir de Jesus Kowal

Sistema desenvolvido em PHP com bd MySql.
Consiste em cadastrar usuarios e senha, e usar a API do PHP para gerar uma criptografia MD5, vista em aula.
Desenvolvi 06 arquivos em PHP (página login, cadastro, conexao com o bd, validação dos dados, index e deslogar), e um banco de dados(com o nome login e uma tabela usuários) que armazena os dados, ja criptografados.
